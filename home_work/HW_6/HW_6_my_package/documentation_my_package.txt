Документация пакета my_package

--math_utils.py--
factorial(n: int) -> int
Вычисляет факториал числа n.

Параметры:
n (int): Неотрицательное целое число, для которого нужно вычислить факториал.

Исключения:
ValueError: если n меньше 0.

Возвращает:
int: Факториал числа n.


gcd(a: int, b: int) -> int
Находит наибольший общий делитель двух чисел a и b.

Параметры:
a (int): Первое число.
b (int): Второе число.

Возвращает:
int: Наибольший общий делитель чисел a и b.


--string_utils.py--
to_upper(text: str) -> str
Преобразует текст в верхний регистр.

Параметры:
text (str): Строка, которую нужно преобразовать.

Возвращает:
str: Строка в верхнем регистре.

del_spaces(text: str) -> str
Удаляет пробелы в начале и в конце строки.

Параметры:
str: Строка, из которой нужно удалить пробелы.
Возвращает:

str: Строка без пробелов в начале и в конце.
